# Budget App

A simple budget management application with GUI built using Python and Tkinter.

## Features

- **Record Income**: Add income transactions with amount, category, and date
- **Record Expense**: Add expense transactions with amount, category, and date
- **View Statistics**: See total income, total expenses, and balance
- **View Transactions**: Display all transactions in a table format
- **Delete Transactions**: Delete single or multiple transactions
- **Data Persistence**: All data is saved in JSON format

## Requirements

- Python 3.7+
- tkinter (included with Python)

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/budget-app.git
cd budget-app
```

2. Run the application:
```bash
python budget_gui.py
```

## File Structure

- `budget_gui.py` - Main GUI application
- `transactions.json` - Data file (auto-created)
